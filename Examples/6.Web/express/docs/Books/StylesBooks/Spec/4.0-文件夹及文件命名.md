### 4.0 文件夹及文件命名

* 4.1 文件夹结构
* 4.2 文件注释

##### 4.1 文件夹结构
```
src
    --common
        --fzs.class.reset.css       // 重置 css 
        --fzs.class.base.css
        
        --fzs.main.css              // 主css
        
        --fzs.grid.css              // 布局 css
        --fzs.layout.css            
        
        --fzs.variable.css          // 变量
        
        --fzs.skin.css              // 皮肤
        
        --fzs.docuemnt.css          // 文档
        
        --fzs.markdown.style.css    // markdown 样式
        
        --fzs.tools.css             // 工具类
        
```


##### 4.2 文件注释

```less
/* 
  @name: 模块或文件名
  @description: 中文说明 
  @author: name 
  @update: name (2016-06-30 15:00) 
*/
```
